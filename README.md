# Portfolio Platform

## How to Run

**Requirements:** XAMPP (or any PHP/MySQL stack) + Composer

1. Clone the repo
   ```bash
   git clone https://github.com/josircalica14/Adv_Webdev_Finals.git
   cd Adv_Webdev_Finals
   ```

2. Start **MySQL** in XAMPP Control Panel

3. Double-click **`setup.bat`**
   - Press Enter three times to accept the defaults
   - It will set everything up and open `http://localhost:8000` automatically

> Keep the terminal window open while using the app.
